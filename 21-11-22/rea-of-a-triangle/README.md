## 3. Area of a Triangle

**Write a function that takes the base and height of a triangle and return its area.**

### Examples

```js
triArea(3, 2) ➞ 3

triArea(7, 4) ➞ 14

triArea(10, 10) ➞ 50
```
#### Notes

- The area of a triangle is: (base * height) / 2
- Don't forget to return the result.
