## Return the First Element in an Array

**Create a function that takes an array containing only numbers and return the first element.**

### Examples

```js
getFirstValue([1, 2, 3]) ➞ 1

getFirstValue([80, 5, 100]) ➞ 80

getFirstValue([-500, 0, 50]) ➞ -500
```
