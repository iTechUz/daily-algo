## Convert Age to Days

**Create a function that takes the age in years and returns the age in days.**

### Examples

```js
calcAge(65) ➞ 23725

calcAge(0) ➞ 0

calcAge(20) ➞ 7300
```
#### Notes
- Don't forget to ```return``` the result.
