## Convert Hours into Seconds

**Write a function that converts ```hours``` into seconds.**

### Examples

```js
howManySeconds(2) ➞ 7200

howManySeconds(10) ➞ 36000

howManySeconds(24) ➞ 86400
```
#### Notes
- Don't forget to ```return``` the result.
