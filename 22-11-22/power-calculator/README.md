## Power Calculator

**Create a function that takes voltage and current and returns the calculated power.**

### Examples

```js
circuitPower(230, 10) ➞ 2300

circuitPower(110, 3) ➞ 330

circuitPower(480, 20) ➞ 9600
```

#### Notes
- Don't forget to ```return``` the result.
